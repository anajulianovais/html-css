Principais formatações

Negrito / Destaque

Nesta frase, há um termo em negrito usando a tag B, uma tag não semântica.

Nesta frase, há um termo em destaque usando a tag strong, uma tag semântica.

Itálico / Ênfase

Nesta frase, há um termo em itálico usando a tag strong, uma tag não semântica.

Nesta frase, há um termo em ênfase usando a tag strong, uma tag semântica.

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande com a tag BIG (obsoleta) e um texto pequeno com a tag SMALL nesse parágrafo.

Texto deletado (tag DEL)

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indiciar que ele foi adicionado depois com a tag INS, uma tag semântica.

Existe também o sublinado com a tag U, que é uma tag não semântica.

Texto sobrescrito (tag SUP)

Para inserir coisas do tipo x2+5x+6

Texto subscrito (tag SUB)

Para inserir coisas do tipo H2O
